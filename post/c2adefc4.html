整体的逻辑框架，需要掌握的基础概念

JVM(Java virtual Machine):Java虚拟机，真正运行Java程序的地方。

核心类库：Java内包含的程序，给程序员用于调用的。

JRE(Java Runtime Environment): Java的运行环境。

JDK(Java Development Kit): Java 的开发工具包

Java的跨平台原理：一次编译，处处可用。 我们的程序只需要开发依次，就可以在各种安装JVM的系统平台上运行。
